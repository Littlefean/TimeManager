# 时间管理软件APP

---



这个项目是一个开发前后端分离的app应用，后端用pythonFlask框架部署在服务器上

前端用html和js来发送网络请求。



## 项目结构

```
D:.
├─.idea
│  └─inspectionProfiles
├─doc	存放一些说明文件
| 
├─myDB	存放自定义数据库生成文件
│  └─jsonDB
|
├─TimeManagerClient	项目的前端工程文件
│  ├─.idea
│  ├─css
│  ├─fonts
│  ├─html
│  ├─img
│  ├─js
│  └─unpackage
|
└─__pycache__
```
